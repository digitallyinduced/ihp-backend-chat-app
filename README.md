# Chat Example App built with IHP Backend

Try it out on https://ihp-backend-chat-app.vercel.app/

## Running locally

```bash
$ npm install
$ npm run watch
$ npm run server
```

The API host is configured in `app.jsx`:

```javascript
initIHPBackend({
    host: 'https://ojomabrabrdiuzxydbgbebztjlejwcey.ihpapp.com'
})
```
